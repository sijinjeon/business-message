## 1. 개요

### 1.1 변경 사항
- **멀티 AI 제공자 지원**: 각 제공자(Gemini, OpenAI, Claude)별 독립적인 키 입력 및 관리
- **번역 설정 통합**: 타겟 언어 및 인라인 번역문 스타일 설정
- **단축키 안내**: 시스템 설정으로 이동하는 링크 제공

---

## 2. UI 구조 (v2.2)

### 2.1 신규 섹션: AI 엔진 및 모델 관리 ⭐
- **메인 엔진 선택**: 현재 활성화할 엔진(Gemini/OpenAI/Claude) 선택
- **제공자별 상세 설정 카드**:
  - **모델 선택 드롭다운**: 각 엔진별 사용 가능한 모델 목록 제공
    - Gemini: `gemini-2.5-pro`, `gemini-2.5-flash`, `gemini-2.0-pro` 등
    - OpenAI: `gpt-5`, `gpt-4o`, `gpt-4o-mini` 등
    - Claude: `claude-4-5-sonnet`, `claude-4-5-haiku`, `claude-3-5-sonnet` 등
  - **API 키 입력 필드**: 각 모델에 적용될 API 키 입력
  - **연결 테스트 버튼**: 선택된 모델과 키로 통신 확인
- **암호화 상태 표시**: 저장된 키가 암호화되어 안전하다는 안내 배지

### 2.2 신규 섹션: 번역 옵션
- **기본 타겟 언어**: 한국어/영어/일본어/중국어 선택
- **인라인 번역 스타일**: 
  - 텍스트 색상 (Color Picker 또는 프리셋)
  - 글자 크기 조정
- **형식 보존 옵션**: 줄바꿈(`\n`) 유지 여부 토글

## 3. 사용자 가이드 및 단축키

### 3.1 단축키 관리 링크
- "Chrome 단축키 설정 열기" 버튼 추가
- 클릭 시 `chrome://extensions/shortcuts`로 연결 (Chrome 제한으로 직접 연결 불가 시 안내 팝업 노출)
