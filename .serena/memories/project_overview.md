# 프로젝트 개요
이 프로젝트는 클립보드에 복사된 문장을 AI를 통해 정중한 비즈니스 표현으로 변환하는 Chrome Extension입니다.

## 주요 기능
- 클립보드 자동 인식 및 변환 준비
- 3가지 톤(Formal, General, Friendly) 변환
- 전문 번역 시스템 지원
- 멀티 AI 엔진 지원 (Gemini, OpenAI, Claude)
- AES-GCM 256-bit 암호화 저장

## 기술 스택
- Frontend: React 18, TypeScript
- UI: shadcn/ui, Tailwind CSS, Lucide React
- Build Tool: Vite 7
- Platform: Chrome Extension Manifest V3
