## 1. 프로젝트 개요

### 1.1 프로젝트 소개

**정중한 문장 도우미**는 한국의 비즈니스 환경에서 사용되는 문장을 AI를 활용하여 세 가지 톤(격식, 일반, 친근)으로 자동 변환해주는 Chrome Extension입니다.

### 1.2 핵심 목표

- **사용자 목표**: 이메일, 메신저 등에서 적절한 톤앤매너를 고민하는 시간을 90% 이상 단축
- **비즈니스 목표**: MVP 출시 3개월 내 일일 활성 사용자(DAU) 1,000명 달성
- **기술 목표**: 팝업 로딩 300ms 이내, API 응답 2초 이하 유지

### 1.3 타겟 사용자

- **주요 타겟**: 2-10년차 사무직 직장인
- **확장 타겟**: 고객 서비스(CS) 및 영업 담당자

### 1.4 프로젝트 상태

✅ **개발 완료 단계 (MVP)**
- 핵심 기능 구현 완료
- 프로덕션 빌드 준비 완료
- Chrome 웹스토어 등록 준비 상태

## 4. 주요 기능 분석

### 4.1 핵심 기능

#### ✅ 1. 클립보드 기반 텍스트 변환

**구현 방식:**
- 팝업 열람 시 자동으로 클립보드 텍스트 읽기
- `navigator.clipboard.readText()` API 사용
- 빈 클립보드 시 수동 입력 가능

**코드 위치:** `src/utils/clipboard.ts`, `src/popup/App.tsx`

**평가:** ⭐⭐⭐⭐⭐ 사용자 경험을 크게 향상시키는 기능

#### ✅ 2. 3가지 톤으로 변환

**톤 타입:**
1. **비즈니스 이메일 (Formal)**: 상사, 외부 고객용 정중한 톤
2. **사내 메신저 (General)**: 동료와의 협업용 부드러운 톤
3. **캐주얼 채팅 (Friendly)**: 친한 동료와의 가벼운 톤

**AI 프롬프트 엔지니어링:**
- 체계적인 가이드라인 정의
- 한국 직장 문화에 맞춤화
- JSON 형식 강제 응답
- 공통 다듬기 원칙 적용

**코드 위치:** `src/utils/api.ts` (createSystemPrompt 함수)

**평가:** ⭐⭐⭐⭐⭐ 프롬프트가 매우 상세하고 체계적

#### ✅ 3. 다중 AI 제공업체 지원

**구현:**
- Gemini, ChatGPT, Claude 지원
- 통일된 인터페이스로 추상화
- 제공업체별 API 호출 로직 분리
- 에러 처리 개별화

**코드 위치:** `src/utils/api.ts`

```typescript
// 통합 함수
export async function convertText(
  userText: string, 
  provider: AIProvider, 
  apiKey: string
): Promise<AIApiResponse>

// 제공업체별 함수
- callGeminiAPI()
- callOpenAIAPI()
- callClaudeAPI()
```

**평가:** ⭐⭐⭐⭐⭐ 확장 가능한 아키텍처

#### ✅ 4. API 키 암호화 저장

**보안 기술:**
- **암호화 알고리즘**: AES-GCM (256-bit)
- **키 생성**: PBKDF2 (100,000 iterations)
- **Salt**: 고정 salt + Extension ID
- **저장소**: Chrome Storage Local (암호화된 상태)

**코드 위치:** `src/utils/crypto.ts`

**평가:** ⭐⭐⭐⭐⭐ 업계 표준 수준의 보안

#### ✅ 5. 자동 클립보드 복사

**기능:**
- 설정에서 활성화/비활성화 가능
- 3가지 톤 중 선호하는 톤 선택 가능
- 변환 완료 시 자동으로 선택된 톤 복사
- 성공 메시지 표시

**코드 위치:** `src/options/Settings.tsx`, `src/popup/App.tsx`

**평가:** ⭐⭐⭐⭐⭐ 워크플로우 최적화

## 5. 코드 구조 및 품질

### 5.1 TypeScript 타입 안정성

**강점:**
- `strict: true` 모드 적용
- 모든 함수에 명확한 타입 정의
- `any` 타입 사용 최소화
- 인터페이스와 타입 체계적 관리

**예시:**
```typescript
export interface AppStorage {
  apiKeys: {
    gemini: string;
    chatgpt: string;
    claude: string;
  };
  dailyUsage: {
    date: string;
    count: number;
  };
  settings: {
    selectedProvider: AIProvider;
    // ...
  };
}
```

**평가:** ⭐⭐⭐⭐⭐ 매우 우수한 타입 안정성

### 5.2 코드 품질

#### 장점:
✅ **명확한 함수명**: `convertText`, `validateApiKey`, `encryptData`
✅ **단일 책임 원칙**: 각 함수가 하나의 역할만 수행
✅ **에러 처리**: try-catch 블록 일관적 적용
✅ **JSDoc 주석**: 주요 함수에 문서화 주석
✅ **React 최적화**: `React.memo`, `useCallback` 활용

#### 개선 가능:
⚠️ 일부 매직 넘버 존재 (예: 글자 수 제한)
⚠️ 테스트 코드 부재

### 5.3 성능 최적화

**최적화 기법:**
1. **React.memo**: 불필요한 리렌더링 방지
2. **useCallback**: 함수 재생성 방지
3. **코드 스플리팅**: 필요 시 동적 import
4. **Terser 압축**: 프로덕션 빌드 최적화
5. **Tree-shaking**: 사용하지 않는 코드 제거

**성능 지표:**
- 팝업 로딩: < 300ms (목표)
- API 응답: < 2초 (목표)
- 번들 크기: 약 68KB (README 기준)
