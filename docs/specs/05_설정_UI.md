## 1. 개요 (v2.3)

설정 페이지는 단순 설정을 넘어 서비스 사용 현황을 한눈에 파악할 수 있는 **사용량 분석 대시보드** 기능을 통합하였습니다.

---

## 2. 메뉴 구성

### 2.1 AI 엔진 및 모델 (최우선 메뉴)
- **엔진 선택**: Gemini, OpenAI, Claude 중 활성 엔진 선택.
- **상세 모델 선택**: 선택한 엔진의 세부 모델을 선택. 모델별 1M 토큰당 예상 비용(KRW/USD) 정보 제공.
- **API 키 관리**: 각 엔진별 키 입력 및 연결 테스트 기능.

### 2.2 일반 및 톤 설정
- **자동화**: 변환 완료 시 자동 클립보드 복사 여부 및 우선 순위 설정.
- **즉시 변환 스타일**: 단축키를 통한 즉시 변환 시 적용할 스타일 지정.

### 2.3 번역 환경 설정
- **기본 언어**: 즉시 번역 시 타겟이 될 기본 언어 설정.

---

## 3. 사용량 분석 대시보드 (신규 ⭐)

### 3.1 KPI 지표
- **총 호출 횟수**: 기간 내 실행된 총 AI 작업 수.
- **총 사용 비용**: 토큰 사용량을 기반으로 계산된 예상 누적 비용 (KRW/USD 병기).
- **평균 응답 속도**: API 호출부터 응답 완료까지의 평균 시간.

### 3.2 데이터 시각화
- **일별 사용량 추이**: 최근 7일간의 사용량을 막대 그래프로 시각화.
- **상세 실행 로그**: 최근 1,000개의 작업 내역(일시, 엔진, 작업 종류, 토큰, 비용)을 테이블로 제공.

---

## 4. 보안 정책 UI
- 모든 API 키 입력 필드는 비밀번호 타입(`type="password"`)으로 처리.
- "Privacy Secured" 배지를 통해 모든 데이터가 로컬에 암호화되어 저장됨을 명시.
