## 1. 개요 (Overview)

- *'정중한 문장 도우미'**는 복잡하고 미묘한 한국의 직장 내 커뮤니케이션을 돕는 AI 기반 크롬 확장 프로그램입니다. 사용자가 웹에서 복사(Ctrl+C)한 텍스트를 클릭 한 번으로 상황에 맞는 세 가지 톤(①격식, ②일반, ③친근)으로 즉시 변환하고, 변환된 문장을 자동으로 클립보드에 저장하여 사용자의 글쓰기 생산성과 커뮤니케이션 품질을 극대화하는 것을 목표로 합니다.

## 2. 목표 (Goal)

- **핵심 목표:** 사용자가 커뮤니케이션 상황에 맞는 적절한 톤앤매너를 고민하는 시간을 90% 이상 단축시킨다.
- **사용자 목표:** 이메일, 메신저 등 디지털 환경에서 문법 오류나 어색한 표현 없이 자신감 있고 전문적인 글쓰기를 지원한다.
- **비즈니스 목표:** MVP 출시 3개월 내 일일 활성 사용자(DAU) 1,000명을 달성하여 제품의 시장성을 검증하고, 향후 프리미엄 기능 도입의 기반을 마련한다.

## 3. 문제 정의 및 해결 방안 (Problem & Solution)

### 3.1. 문제 정의 (Problem)

- **과도한 시간 소요:** 직급, 상황, 대상에 따라 적절한 존댓말과 격식을 갖춘 문장을 작성하는 데 불필요한 시간과 정신적 에너지가 소모됩니다.
- **커뮤니케이션 오류:** 급하게 작성한 글에서 발생하는 어색한 표현이나 문법 오류가 개인과 조직의 전문성에 대한 신뢰도를 저하시킬 수 있습니다.
- **반복 작업의 비효율:** 회의 요청, 감사 인사 등 유사한 목적의 문장을 매번 새로 작성하는 것은 비효율적인 반복 작업입니다.

### 3.2. 해결 방안 (Solution)

- **즉각적인 AI 변환:** 클립보드에 복사된 텍스트를 AI가 즉시 분석하여, '격식있는 비즈니스', '일반적인 동료', '친근한' 톤의 세 가지 대안을 제시합니다.
- **원클릭 워크플로우:** 확장 프로그램 아이콘 클릭 한 번으로 문장 변환부터 클립보드 복사까지의 전 과정을 자동화하여 작업 흐름의 끊김을 최소화합니다.
- **신뢰성 있는 결과:** OpenAI의 강력한 언어 모델을 기반으로 자연스럽고 정확한 문장을 생성하여 사용자가 별도의 교정 없이 바로 사용할 수 있도록 합니다.

## 4. 핵심 사용자 (Target Audience)

- **주요 타겟 (Primary): 2-10년차 사무직 직장인**
    - 이메일, 슬랙, 팀즈 등 다양한 채널로 내/외부 커뮤니케이션이 잦습니다.
    - 상사에게 보고하거나 외부 고객에게 연락할 때 적절한 존댓말과 표현에 대한 고민이 많습니다.
- **확장 타겟 (Secondary): 고객 서비스(CS) 및 영업 담당자**
    - 고객과의 대화에서 일관되고 전문적인 톤을 유지하는 것이 매우 중요합니다.
    - 표준화된 응대 가이드라인을 빠르고 쉽게 적용할 필요가 있습니다.

## 5. 제품 범위 및 기능: Phase 1 (MVP)

MVP 단계에서는 핵심 가치인 '클립보드 기반의 빠른 문장 변환'에 집중합니다.

### 5.1. 핵심 기능: 클립보드 기반 문장 변환

- **자동 텍스트 인식:** 사용자가 `Ctrl+C`로 텍스트를 복사한 후 확장 프로그램 팝업을 열면, 클립보드의 텍스트를 자동으로 감지하여 입력창에 표시합니다.
- **AI 기반 톤 변환:** OpenAI (ChatGPT) API를 호출하여 원본 텍스트를 아래 3가지 톤으로 변환한 결과를 제공합니다.
    1. **격식있는 비즈니스 (기본값):** 보고서, 외부 이메일 등에 적합한 정중한 톤
    2. **일반적인 동료:** 팀 내 메신저 등 수평적인 소통에 적합한 톤
    3. **친근한:** 사내 동료와의 가벼운 대화에 적합한 톤
- **결과 자동 복사:** 3가지 결과 중 기본값인 '격식있는 비즈니스' 톤의 문장을 자동으로 사용자의 클립보드에 복사하여 `Ctrl+V`로 즉시 사용할 수 있게 합니다.

### 5.2. UI/UX 기능

- **결과 카드:** 각 톤별 변환 결과를 명확히 구분된 UI 카드 형태로 표시합니다.
- **개별 복사 버튼:** 각 결과 카드 우측에 '복사하기' 아이콘을 배치하여 원하는 톤의 결과만 선택적으로 복사할 수 있습니다.
- **재생성 버튼:** 현재의 원본 텍스트로 새로운 AI 추천 문장을 받아볼 수 있는 '다시 생성' 기능을 제공합니다.
- **수동 입력:** 클립보드가 비어있거나 다른 문장을 테스트하고 싶을 경우, 팝업 내 텍스트 영역에 직접 타이핑하여 변환할 수 있습니다.
- **상태 표시:** AI가 결과를 생성하는 동안 로딩 애니메이션을 표시하여 사용자가 시스템의 작동 상태를 인지할 수 있도록 합니다.

### 5.3. 설정 기능

- **API 키 관리:** 사용자가 자신의 OpenAI API 키를 입력하고 저장하는 설정 페이지를 제공합니다.
- **API 키 유효성 검증:** 입력된 API 키가 유효한지 확인할 수 있는 '연결 테스트' 버튼을 제공합니다.
- **사용량 제한:** 무료 사용자를 위한 일일 사용 횟수(5회)를 제한하고, 팝업 내에 남은 횟수를 표시합니다. (사용량 카운트는 로컬 기준)

## 6. 사용자 플로우 및 시나리오 (User Flow & Scenarios)

### 6.1. 기본 사용 플로우

`사용자: 이메일 작성 중 "이번 주까지 꼭 전달 부탁드립니다." 문장을 정중하게 바꾸고 싶다.`

1. **[Action]** 사용자가 "이번 주까지 꼭 전달 부탁드립니다." 텍스트를 마우스로 드래그하여 `Ctrl+C`로 복사합니다.
2. **[Action]** 브라우저 툴바의 '정중한 문장 도우미' 아이콘을 클릭합니다.
3. **[System]** 팝업창이 열리면서 복사된 텍스트를 자동으로 인식하고, 로딩 상태를 표시하며 OpenAI API로 변환을 요청합니다.
4. **[System]** 1~3초 내에 3가지 톤의 변환 결과를 팝업에 표시합니다.
    - **격식:** "업무에 참고할 수 있도록, 가능하시다면 이번 주 내로 자료를 전달해주시면 대단히 감사하겠습니다."
    - **일반:** "이번 주 안에 전달해주시면 업무 진행에 큰 도움이 될 것 같아요."
    - **친근:** "이번 주까지 꼭 부탁드려요! 화이팅!"
5. **[System]** '격식' 톤의 결과가 자동으로 클립보드에 복사됩니다.
6. **[Action]** 사용자는 이메일 작성창으로 돌아가 `Ctrl+V`로 변환된 문장을 붙여넣습니다.

### 6.2. 예외 시나리오

- **클립보드가 비어있을 경우:** 팝업에 "변환할 텍스트를 복사하거나, 여기에 직접 입력해주세요." 라는 안내 메시지와 함께 빈 입력창을 표시합니다.
- **API 호출 실패 시:** "AI 서버 응답에 실패했습니다. 잠시 후 '다시 생성' 버튼을 눌러주세요." 라는 오류 메시지를 표시합니다.
- **일일 사용량 초과 시:** 팝업에 "오늘의 무료 사용량을 모두 사용하셨습니다." 라는 안내 메시지를 표시합니다.

## 7. 기술 요구사항 및 스택 제안 (Technical Requirements & Stack Proposal)

### 7.1. 프론트엔드 (Chrome Extension)

- **Manifest Version:** **V3** (최신 표준으로 보안 및 성능 강화)
- **개발 언어:** **TypeScript** (JavaScript 기반, 코드 안정성 및 유지보수성 향상)
- **UI 프레임워크:** **React.js** (컴포넌트 기반으로 빠르고 효율적인 UI 개발 가능, 생태계가 넓어 라이브러리 활용에 용이)
- **필요 권한 (Permissions):**
    - `clipboardRead`: 사용자의 클립보드 내용을 읽기 위한 권한
    - `clipboardWrite`: 변환된 텍스트를 클립보드에 쓰기 위한 권한
    - `storage`: 사용자의 API 키 및 설정값을 안전하게 저장하기 위한 권한
- **데이터 저장:** **Chrome Storage API** (`storage.local`)를 사용하여 사용자의 API 키, 사용량 등의 데이터를 브라우저 내 로컬 환경에 안전하게 저장합니다.

### 7.2. API 연동

- **AI 서비스:** Google Gemini 2.5 Flash Lite Model 사용
- **핵심 프롬프트:** 한국어 비즈니스 및 일상 커뮤니케이션 문맥에 최적화된 시스템 프롬프트를 설계하여 일관되고 높은 품질의 결과물을 생성하도록 합니다.
    - *예시: "당신은 한국 직장 문화에 대한 이해도가 매우 높은 커뮤니케이션 전문가입니다. 다음 텍스트를 '격식있는 비즈니스', '일반적인 동료', '친근한' 세 가지 톤으로 변환해주세요."*
- **에러 핸들링:** 네트워크 오류, API 키 오류, 서버 과부하 등 다양한 API 응답 코드에 대한 안정적인 예외 처리 로직을 구현합니다.

### 7.3. 보안 및 개인정보

- **API 키 암호화:** 사용자가 입력한 API 키는 `Chrome Storage API`에 저장 시 기본적인 암호화 처리를 권장합니다.
- **데이터 처리:** 사용자가 입력한 텍스트는 오직 OpenAI API 호출 목적으로만 실시간으로 사용되며, **별도의 외부 서버에 절대 저장하거나 전송하지 않습니다.**
- **사용량 추적:** 일일 사용량은 서버가 아닌 사용자의 브라우저 로컬 스토리지에서만 카운트하여 개인정보 침해 소지를 원천 차단합니다.

## 8. 제품 로드맵 (Product Roadmap)

### Phase 2: 편의 기능 확장 (MVP 출시 후 3개월)

- **텍스트 스니펫:** 자주 사용하는 인사말, 업무 요청 문구 등을 저장하고 `/인사`와 같은 단축키로 빠르게 불러오는 기능
- **고급 변환 옵션:** 문장의 길이를 조절하거나, 특정 키워드를 포함/제외하는 등의 고급 설정 추가
- **다중 AI 모델 지원:** 사용자가 Google Gemini, Anthropic Claude 등 다른 AI 모델을 선택할 수 있는 옵션 제공

### Phase 3: 서비스 고도화 (MVP 출시 후 6개월)

- **웹페이지 통합:** Gmail, Slack 등 특정 웹사이트의 입력창에서 텍스트를 작성하면 실시간으로 변환 제안을 보여주는 인라인(In-line) 기능
- **팀 기능:** 팀 단위로 공통 스니펫을 공유하고 관리하는 기능 (B2B 모델)
- **프리미엄 구독 모델:** 무제한 사용, 고급 기능, 우선 지원 등을 포함하는 유료 구독 플랜 도입
