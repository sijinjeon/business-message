## 1. 개요

### 1.1 구조 변경
팝업은 이제 **탭(Tab) 시스템**을 사용하여 두 가지 주요 기능을 제공합니다.
- **Tab 1: 📝 정중한 문장** (기존 기능)
- **Tab 2: 🌐 전문 번역** (신규 기능)

---

## 2. UI 구조 (v2.2)

### 2.1 탭 레이아웃
```
┌─────────────────────────────────────────────┐
│  [ 📝 정중한 문장 ]  [ 🌐 번역 ]             │  ← 상단 탭 전환 (48px)
├─────────────────────────────────────────────┤
│  Common Header                              │
│  Provider Selector                          │
├─────────────────────────────────────────────┤
│  Active Tab Content (Input & Result)        │  ← 선택된 탭에 따라 변경
└─────────────────────────────────────────────┘
```

---

## 3. 탭별 상세 명세

### 3.1 번역 탭 (Translation Tab)
- **대상 언어 선택**: 한국어, 영어, 일본어, 중국어 선택 드롭다운
- **수동 입력창**: 텍스트를 직접 입력하거나 클립보드 내용을 번역
- **결과 카드**: 번역된 단일 텍스트를 표시하고 복사 버튼 제공

### 3.2 상태 관리 통합
- 팝업이 닫혀도 마지막으로 활성화된 탭 정보를 스토리지에 저장하여 다음 오픈 시 복원합니다.
- 입력값(`inputText`)은 탭 간 이동 시에도 유지되어 연속적인 작업이 가능하게 합니다.
