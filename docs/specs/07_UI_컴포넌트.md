## 1. 개요

### 1.1 UI 라이브러리

**shadcn/ui**: Radix UI + Tailwind CSS 기반 컴포넌트 라이브러리

**선택 이유**:
- ✅ 복사-붙여넣기 방식 (dependency 최소화)
- ✅ 완전한 커스터마이징 가능
- ✅ 접근성 (a11y) 기본 제공
- ✅ TypeScript 완벽 지원
- ✅ Tailwind CSS와 완벽 통합

### 1.2 설치된 컴포넌트

```bash
src/components/ui/
├── alert.tsx           # 알림/메시지
├── button.tsx          # 버튼
├── card.tsx            # 카드 컨테이너
├── checkbox.tsx        # 체크박스
├── input.tsx           # 입력 필드
├── select.tsx          # 선택 드롭다운
├── skeleton.tsx        # 로딩 스켈레톤
└── textarea.tsx        # 텍스트 영역
```

## 2. shadcn/ui 컴포넌트

### 2.1 Button

**파일**: `src/components/ui/button.tsx`

**Variants**:
```typescript
const buttonVariants = cva(
  "inline-flex items-center justify-center ...",
  {
    variants: {
      variant: {
        default: "bg-primary text-primary-foreground hover:bg-primary/90",
        destructive: "bg-destructive text-destructive-foreground ...",
        outline: "border border-input bg-background hover:bg-accent ...",
        secondary: "bg-secondary text-secondary-foreground ...",
        ghost: "hover:bg-accent hover:text-accent-foreground",
        link: "text-primary underline-offset-4 ..."
      },
      size: {
        default: "h-10 px-4 py-2",
        sm: "h-9 rounded-md px-3",
        lg: "h-11 rounded-md px-8",
        icon: "h-10 w-10"
      }
    }
  }
)
```

**사용 예시**:
```tsx
// Primary 버튼
<Button>저장</Button>

// Secondary 버튼
<Button variant="secondary">연결 테스트</Button>

// Ghost 버튼 (아이콘만)
<Button variant="ghost" size="sm">
  <Settings className="h-4 w-4" />
</Button>

// Outline 버튼
<Button variant="outline">클립보드 읽기</Button>
```

### 2.2 Card

**파일**: `src/components/ui/card.tsx`

**구성 요소**:
- `Card`: 카드 컨테이너
- `CardHeader`: 헤더 영역
- `CardTitle`: 제목
- `CardDescription`: 설명
- `CardContent`: 본문
- `CardFooter`: 푸터

**스타일**:
```
배경: hsl(var(--card))
텍스트: hsl(var(--card-foreground))
테두리: 1px solid hsl(var(--border))
둥근 모서리: 0.5rem
그림자: 없음 (플랫 디자인)
```

**사용 예시**:
```tsx
<Card>
  <CardHeader>
    <CardTitle>Google Gemini API 키 설정</CardTitle>
  </CardHeader>
  <CardContent>
    {/* 내용 */}
  </CardContent>
</Card>
```

### 2.3 Input & Textarea

**Input**: 한 줄 입력 (API 키 등)
```tsx
<Input
  type="password"
  value={apiKey}
  onChange={(e) => setApiKey(e.target.value)}
  placeholder="API 키를 입력하세요"
  className="font-mono"
/>
```

**Textarea**: 여러 줄 입력 (변환할 텍스트)
```tsx
<Textarea
  value={text}
  onChange={(e) => setText(e.target.value)}
  placeholder="텍스트를 입력하세요"
  className="min-h-[120px] resize-none"
/>
```

### 2.4 Select

**구성 요소**:
- `Select`: 컨테이너
- `SelectTrigger`: 트리거 버튼
- `SelectValue`: 선택된 값 표시
- `SelectContent`: 드롭다운 내용
- `SelectItem`: 각 옵션

**사용 예시**:
```tsx
<Select value={value} onValueChange={onChange}>
  <SelectTrigger>
    <SelectValue placeholder="선택하세요" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="option1">옵션 1</SelectItem>
    <SelectItem value="option2">옵션 2</SelectItem>
  </SelectContent>
</Select>
```

### 2.5 Alert

**Variants**:
- `default`: 일반 알림 (파란색)
- `destructive`: 에러 알림 (빨간색)

**사용 예시**:
```tsx
{/* 성공 메시지 */}
<Alert>
  <AlertDescription>
    ✅ 설정이 저장되었습니다!
  </AlertDescription>
</Alert>

{/* 에러 메시지 */}
<Alert variant="destructive">
  <AlertDescription>
    ❌ API 키가 유효하지 않습니다.
  </AlertDescription>
</Alert>
```

### 2.6 Skeleton

**로딩 스켈레톤**: 콘텐츠 로딩 중 표시

```tsx
<Skeleton className="h-32" />  {/* 128px 높이 */}
<Skeleton className="h-20 w-full" />  {/* 80px 높이, 전체 너비 */}
```

**애니메이션**:
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}

animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;
```

### 2.7 Checkbox

**사용 예시**:
```tsx
<div className="flex items-center space-x-2">
  <Checkbox
    id="auto-copy"
    checked={checked}
    onCheckedChange={setChecked}
  />
  <label htmlFor="auto-copy" className="text-sm cursor-pointer">
    자동 복사 활성화
  </label>
</div>
```

```

```

## 5. 아이콘 시스템

### 5.1 lucide-react

**설치된 아이콘**:
```typescript
import {
  Settings,      // 설정
  RotateCw,      // 다시 생성 (회전)
  Clipboard,     // 클립보드
  Copy,          // 복사
  Check,         // 체크 (복사 완료)
  Loader2,       // 로딩 스피너
  CheckCircle,   // 성공
  XCircle        // 에러
} from 'lucide-react'
```

**스타일**:
- 크기: 16px (`h-4 w-4`)
- 두께: 1.5px (기본값)
- 색상: 부모 컴포넌트 따름

**사용 예시**:
```tsx
<Settings className="h-4 w-4" />
<RotateCw className="h-4 w-4 animate-spin" />
<Check className="h-4 w-4 text-green-600" />
```
