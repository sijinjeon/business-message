# 기능 상세 스펙 문서 - 04. 팝업 UI

> **문서 버전**: 2.3 (2026 Update)
> **작성일**: 2026-01-01
> **업데이트**: 2026-01-01 - 전문 번역 탭 및 자동 실행 로직 반영
> **담당 모듈**: `src/popup/`

---

## 1. 개요 (v2.3)

사용자 편의성을 위해 800x600px의 확장된 레이아웃을 채택하였으며, 좌측 입력/우측 결과의 2열 구조로 개편되었습니다.

---

## 2. 레이아웃 구조

### 2.1 메인 레이아웃 (2-Column)
- **좌측 섹션 (340px)**: 텍스트 입력 영역. 클립보드 붙여넣기 버튼, 탭 전환 버튼(톤 변환/전문 번역), 번역 언어 선택기 포함.
- **우측 섹션 (잔여 영역)**: 결과 표시 영역. 선택된 탭에 따라 톤별 결과 카드 또는 번역 결과가 표시됨.

---

## 3. 핵심 기능 UI

### 3.1 탭 시스템
- **톤 변환 (Tone Tab)**: 3가지 톤(격식, 일반, 친근)의 결과를 카드로 표시. 각 카드는 복사 및 개별 재생성 버튼 지원.
- **전문 번역 (Translation Tab)**: 타겟 언어 선택 후 번역 결과를 하나의 통합 결과 영역에 표시.

### 3.2 자동 실행 및 클립보드 감지
- **자동 감지**: 팝업 오픈 시 클립보드 내용을 자동으로 입력창에 채움.
- **명령어 대응**: 단축키(예: `_execute_action`)를 통해 팝업이 열린 경우, 5초 이내의 명령어라면 즉시 AI 처리를 시작함.

### 3.3 상태 표시 (Status Feedback)
- 상단 헤더에 성공/에러 메시지를 세련된 배지 형태로 일시 표시 (2초 후 소멸).
- AI 응답 대기 시 그라데이션 스피너와 함께 로딩 상태 표시.

---

## 4. 주요 컴포넌트 스펙

- **ResultCard**: 톤 변환 결과를 담는 컴포넌트로, 설정된 '우선 복사 스타일'에 따라 하이라이트 효과 적용.
- **TextInput**: 최대 5,000자 지원, 글자 수 카운터 및 비우기 기능 포함.
