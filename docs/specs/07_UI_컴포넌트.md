# 기능 상세 스펙 문서 - 07. UI 컴포넌트

> **문서 버전**: 2.3 (2026 Update)
> **작성일**: 2026-01-01
> **업데이트**: 2026-01-01 - 2026년 기준 UI 가이드라인 및 컴포넌트 라이브러리 최신화
> **담당 모듈**: `src/components/ui/`, `src/popup/components/`

---

## 1. 개요 (v2.3)

본 프로젝트는 **shadcn/ui**를 기반으로 한 일관된 디자인 시스템을 유지합니다. 2026년 업데이트에서는 더 정밀한 인터랙션과 접근성(a11y) 기준을 준수하도록 개선되었습니다.

---

## 2. 컴포넌트 라이브러리 (shadcn/ui)

현재 사용 중인 핵심 UI 컴포넌트는 다음과 같습니다:
- **Button, Input, Textarea**: 폼 입력 및 액션 처리.
- **Select, Checkbox**: 엔진 선택 및 설정 전환.
- **Card, Alert**: 결과 표시 및 알림 피드백.
- **Skeleton**: 비동기 데이터 로딩 중 시각적 피드백 제공.

---

## 3. 커스텀 비즈니스 컴포넌트

- **ResultCard**: 톤 변환 결과를 시각화하며, 설정된 '자동 복사 톤'에 따른 강조 표시 및 개별 재생성(Retry) 로직을 포함합니다.
- **TextInput**: 글자 수 실시간 카운팅(최대 5,000자) 및 클립보드 연동 기능을 내장한 전용 입력창입니다.
- **DashboardKPI**: 설정 페이지 대시보드에서 사용되는 수치 시각화 컴포넌트입니다.

---

## 4. 디자인 시스템 가이드

- **색상**: `zinc` 계열의 모노톤을 베이스로 하여 전문적인 비즈니스 톤을 유지합니다. 성공/에러 시에는 `green`/`red` 포인트를 사용합니다.
- **타이포그래피**: `Pretendard` 또는 시스템 폰트를 사용하여 가독성을 최우선으로 합니다.
- **애니메이션**: `framer-motion` 또는 `tailwindcss-animate`를 활용한 자연스러운 탭 전환 및 토스트 노출 효과를 적용합니다.
